# Quickstart

Helios Engine is a library for creating video-ready animations using web technologies. It allows you to build compositions using standard HTML, CSS, and Canvas (or frameworks like React and Vue) and render them to high-quality video.

## The Fastest Way to Start

The quickest way to get started is by using the Helios CLI to scaffold a new project.

### Using the CLI

1.  Initialize a new project:
    ```bash
    npx helios init <project-name>
    ```

2.  Navigate to the directory and install dependencies:
    ```bash
    cd <project-name>
    npm install
    ```

3.  Start the development server and open Helios Studio:
    ```bash
    npm run dev
    ```
    Open your browser to the URL shown to access Helios Studio.
