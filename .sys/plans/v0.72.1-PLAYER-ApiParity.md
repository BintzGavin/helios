#### 1. Context & Goal
- **Objective**: Implement missing `HTMLVideoElement` properties (`width`, `height`, `playsInline`) and methods (`fastSeek`) in `HeliosPlayer` to improve parity with the standard media API.
- **Trigger**: Gap analysis against `AGENTS.md` goal of "Feature parity with standard HTMLMediaElement".
- **Impact**: Improves compatibility with third-party video libraries and wrappers that expect a standard video element interface, reducing integration friction.

#### 2. File Inventory
- **Modify**: `packages/player/src/index.ts` (Add getters/setters for width, height, playsInline, and fastSeek method)
- **Modify**: `packages/player/src/api_parity.test.ts` (Add unit tests for new properties)
- **Read-Only**: `packages/player/src/features/*`

#### 3. Implementation Spec
- **Architecture**: Extend `HeliosPlayer` Web Component (extending `HTMLElement`) to include properties found on `HTMLVideoElement` interface.
- **Pseudo-Code**:
    - **width**:
        - Getter: Parse `width` attribute as integer (base 10). Return 0 if missing/invalid.
        - Setter: Set `width` attribute to string value of input.
    - **height**:
        - Getter: Parse `height` attribute as integer (base 10). Return 0 if missing/invalid.
        - Setter: Set `height` attribute to string value of input.
    - **playsInline**:
        - Getter: Return true if `playsinline` attribute exists, false otherwise.
        - Setter: Add `playsinline` attribute if true, remove if false.
    - **fastSeek(time)**:
        - Logic: Call `this.currentTime = time` (delegating to existing seek logic).
- **Public API Changes**:
    - `width: number`
    - `height: number`
    - `playsInline: boolean`
    - `fastSeek(time: number): void`
- **Dependencies**: None.

#### 4. Test Plan
- **Verification**: `npx vitest run packages/player/src/api_parity.test.ts`
- **Success Criteria**:
    - `width` and `height` properties correctly reflect attributes and vice versa.
    - `playsInline` property correctly toggles attribute.
    - `fastSeek` updates `currentTime`.
- **Edge Cases**:
    - Setting width/height to non-numeric strings (should be handled by standard attribute behavior, `parseInt` handles parsing).
    - `fastSeek` with time outside bounds (handled by `seek` logic).
